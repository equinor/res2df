wcon

This module extracts information from WCONHIST, WCONINJE, WCONINJH and WCONPROD from a .DATA file.

from res2df import wcon, ResdataFiles

resdatafiles = ResdataFiles("MYDATADECK.DATA")
dframe = wcon.df(resdatafiles)
