ecl2df.vfp._vfp

Extract the VFPPROD/VFPINJ data from an Eclipse (input) deck as Pandas Dataframes

Data can be extracted from a full Eclipse deck or from individual files. Supports output both in csv format as a pandas DataFrame or in pyarrow and pyarrow.table

ecl2df.vfp._vfp.basic_data(deck, keyword='VFPPROD', vfpnumbers_str=None)[source]

Produce a dictionary with basic data for an Eclipe VFPPROD/VFPINJ. Dictionary returned contains items for liftcuve tables as simple datatypes Required keys in dictionary for VFPPROD and VFPINJ can be found in BASIC_DATA_KEYS in _vfpprod and _vfpinj.

Parameters

  • deck (Union[str, EclFiles, Deck]) – Eclipse deck or string with deck

  • keyword (str) – VFP table type, i.e. ‘VFPPROD’ or ‘VFPINJ’

  • vfpnumbers_str (Optional[str]) – String with list of vfp table numbers to extract. Syntax “[0,1,8:11]” corresponds to [0,1,8,9,10,11].

Return type

List[Dict[str, Any]]

Produce a string defining all VFPPROD/VFPINJ Eclipse input from a dataframe

All data for the keywords VFPPROD/VFPINJ will be returned.

Parameters

  • dframe (DataFrame) – Dataframe containing complete description of VFPPROD/VFPINJ input

  • keyword (str) – Keywords to include, i.e. ‘VFPPROD’ or ‘VFPINJ’

  • comments (Optional[Dict[str, str]]) – comments: Dictionary indexed by keyword with comments to be included pr. keyword. If a key named “master” is present it will be used as a master comment for the outputted file.

  • filename (Optional[str]) – If supplied, the generated text will also be dumped to file.

str
