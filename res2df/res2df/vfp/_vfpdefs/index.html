res2df.vfp._vfpdefs

Some definitions and parameters used to define VFPPROD and VFPINJ keywords in Eclipse. This includes definitions of rates, thp, wfr (water fractions), gfr (gas fractions), alq (artificial-lift-quantities), units and so on. Used for consistency check in IO routines for VFPPROD and VFPINJ keywords in res2df.
