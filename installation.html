InstallationÔÉÅ

Internally in Equinor, res2df is distributed through Komodo and nothing is needed besides activating Komodo.

See https://fmu-docs.equinor.com/docs/komodo/equinor_komodo_usage.html for Komodo instructions.

On Linux computers outside Equinor, res2df should be installed from https://pypi.org:

pip install res2df

For MacOS, the OPM dependency is not available from pypi, and OPM must be compiled manually.
