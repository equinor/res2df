res2df.fipreports

Extract FIP region reports from PRT file

res2df.fipreports.df(prtfile, fipname='FIPNUM')[source]

Parses a PRT file from and finds FIPXXXX REGION REPORT blocks and organizes those numbers into a dataframe

Each row in the dataframe represents one parsed line in the PRT file, with DATE and region index added.

Parameters:

  • prtfile (Union[str, ResdataFiles]) – filename (PRT) or a ResdataFiles object

  • fipname (str) – The name of the regport regions, FIPNUM, FIPZON or whatever Max length of the string is 8, the first three characters must be FIP, and the next 3 characters must be unique for a given .DATA file.

Return type:

DataFrame
